## What is VQS?

VQS is a service or bot landing page built with Next.js (pages router) and TypeScript. It includes API routes alongside the main landing page, making it a proper full-stack Next.js setup rather than a plain static site.

---

## Features

- **Next.js pages router** — file-based routing with SSR support
- **TypeScript** — type-safe throughout, including components and API routes
- **API routes** — backend endpoints under `/api/`
- **Fast page loads** — server-side rendering out of the box

---

## How to Use

**Requirement:** Node.js 18+

```bash
npm install
npm run dev
# Opens at http://localhost:3000
```

**Production build:**
```bash
npm run build
npm start
```
